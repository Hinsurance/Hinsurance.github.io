1.

var arr = [];
arr[‘a’]=1;
console.log(arr.length);//A 0
arr[‘2’]=2;
console.log(arr.length);//B 3
arr.length = 0;
console.log(arr);//C [a: 1]
问A、B、C处的输出结果

A：给数组添加命名属性，数组的length值并未发生变化
B：向数组添加一个属性，但是属性名“看起来”像一个数字，那它会变成一个数值下标
C：设置数组的length可以改变数组的大小，但是对命名属性没影响。设置值小于当前值时，数组被截断；大于时，数组增大，添加的新元素为undefined。
参考：《你不知道的javascript》 P 109
